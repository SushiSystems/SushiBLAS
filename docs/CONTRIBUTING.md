# Contributing to SushiBLAS

SushiBLAS is currently a personal project developed by Mustafa Garip. To maintain a unified vision and ensure the highest performance standards, I am **not accepting external code contributions or Pull Requests (PRs)** at this stage.

## How You Can Help

While I am not accepting code, you are still welcome to:

1.  **Report Bugs**: If you find an issue, please report it via GitHub Issues.
2.  **Suggest Ideas**: Feature requests and architectural discussions are welcome in the Issues section.

## Development Standards

I follow a strict [Style Guide](STYLE_GUIDE.md) to maintain consistency. If you are interested in how the project is built, please refer to the documentation in the `docs/` and `scripts/` folders.
